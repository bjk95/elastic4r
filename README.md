# elastic4r

## Command line interface

Required fields query: the search term

USAGE:
    elastic4r [OPTIONS] <query>

FLAGS:
    -h, --help       
            Prints help information

    -V, --version    
            Prints version information


OPTIONS:
    -c, --config <config-path>    
             [default: config/example-conf-brad.yaml]


ARGS:
    <query>
